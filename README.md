AnadBackend is a modern TypeScript-based backend API built with Express.js, Prisma ORM, and JWT authentication. The project provides a scalable and secure foundation for full-stack applications, featuring user authentication, refresh token handling, protected routes, and database integration with Prisma.

The backend follows clean architecture principles and includes secure password hashing with bcrypt, HTTP-only cookie authentication, middleware-based route protection, and RESTful API design. It is structured for maintainability, scalability, and production-ready deployment.

Key Features:

* Express.js REST API
* TypeScript support
* Prisma ORM integration
* JWT Access & Refresh Authentication
* Secure HTTP-only cookie handling
* Role-based authentication structure
* Middleware-based route protection
* Modular and scalable folder architecture
* MySQL/PostgreSQL compatible via Prisma
* Environment-based configuration

Tech Stack:
TypeScript, Node.js, Express.js, Prisma ORM, JWT, bcryptjs, Cookie Parser, MySQL/PostgreSQL
